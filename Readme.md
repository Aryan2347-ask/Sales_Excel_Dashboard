🚀 Project Overview

This project is an **Interactive Sales Performance Dashboard** developed using **Microsoft Excel**.
The dashboard provides deep insights into:

* 📈 Sales Performance
* 🌍 Regional Analysis
* 👥 Customer Analysis
* 📦 Product Performance
* 🎯 Target vs Actual Sales
* 📅 Monthly & Weekly Trends

The dashboard is designed to help businesses monitor KPIs, identify high-performing regions/products, and make data-driven decisions.

📂 Dataset Information

The dataset contains:

| Column Name    | Description            |
| -------------- | ---------------------- |
| DATE           | Sales transaction date |
| CUSTOMER NAME  | Name of customer       |
| PRODUCT        | Product sold           |
| UNIT PRICE ($) | Product price          |
| QUANTITY       | Quantity purchased     |
| ACTUAL SALES   | Final sales amount     |
| COUNTRY        | Customer country       |
| REGION         | Sales region           |
| MONTH          | Month of transaction   |
| WEEK           | Week number            |
